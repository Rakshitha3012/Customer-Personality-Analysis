# Customer-Personality-Analysis
Data Cleaning and Preprocessing using python on Customer Personality Analysis Dataset.

Customer Personality Analysis - Data Cleaning

Objective

The objective of this project is to clean and preprocess the Customer Personality Analysis dataset using Python and Pandas.

Dataset

- Dataset: Customer Personality Analysis
- Source: Kaggle

Tools Used

- Python 3.13
- Pandas Library

Data Cleaning Steps

1. Loaded the dataset using Pandas.
2. Checked the dataset shape and data types.
3. Identified missing values.
4. Filled missing values in the Income column using the median.
5. Removed duplicate records.
6. Converted the Dt_Customer column to datetime format.
7. Standardized text values in categorical columns.
8. Saved the cleaned dataset.

Output Files

- marketing_campaign.csv
- cleaned_marketing_campaign.csv
- T1.py
- README.md

Conclusion

The dataset was successfully cleaned and preprocessed. Missing values were handled, duplicate records were removed, and the cleaned dataset was saved for further analysis.
